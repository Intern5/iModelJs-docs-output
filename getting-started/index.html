@bentley/ninezone-react-components

React components for 9 zone UI and other purposes.

🔗Usage

🔗Installation

npm install @bentley/ninezone-react-components

🔗Simple Example

import { AppButton } from "@bentley/ninezone-react-ui-components";

🔗Contributing

Please push code and pull requests to the Git repository.

🔗Coding Guidelines

Look at the existing code and try to keep your code similar.

  • Every Sass file should have a :className { ... } block defined for the root element of the component.

  • Import the Sass classnames with: import * as css from "./YourComponent.scss";

  • Every component should accept className?: string and style?: React.CSSProperties props.

  • Components that accept one or a list of children should use the children?: React.ReactNode prop.

🔗Advice

  • Use classnames function to create className strings for the elements.

  • Use const className inside render for the root element className value.
