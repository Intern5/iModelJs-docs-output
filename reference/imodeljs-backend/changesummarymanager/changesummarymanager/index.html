🔗ChangeSummaryManager

Class to extract change summaries for a briefcase.

🔗Methods

🔗attachChangeCache StaticPublic

attachChangeCache(iModel: IModelDb): void

Attaches the Changes cache file to the specified iModel if it hasn't been attached yet. A new Changes cache file will be created for the iModel if it hasn't existed before.

throws IModelError

Parameter Type Description
iModel IModelDb iModel to attach the Changes cache file to

Returns: void

🔗detachChangeCache StaticPublic

detachChangeCache(iModel: IModelDb): void

Detaches the ECChanges cache file from the specified iModel.

throws IModelError in case of errors, e.g. if no ECChanges cache was attached before.

Parameter Type Description
iModel IModelDb iModel to detach the ECChanges cache file to

Returns: void

🔗extractChangeSummaries StaticPublic

extractChangeSummaries(iModel: IModelDb, options: ChangeSummaryExtractOptions): Promise

Extracts change summaries from the specified iModel. Change summaries are extracted from the specified startChangeSetId up through the change set the iModel was opened with. If startChangeSetId is undefined, the first changeset will be used.

throws IModelError if the iModel is standalone,r was not opened in readwrite mode.

Parameter Type Description
iModel IModelDb iModel to extract change summaries for. The iModel must not be a standalone iModel, and it must be opened

with OpenMode.ReadWrite. Note: The method moves the history of the iModel back to the specified start changeset. After the extraction has completed, the iModel is moved back to the original changeset. options | ChangeSummaryExtractOptions | Extraction options

Returns: Promise

🔗isChangeCacheAttached StaticPublic

isChangeCacheAttached(iModel: IModelDb): boolean

Determines whether the Changes cache file is attached to the specified iModel or not

Parameter Type Description
iModel IModelDb iModel to check whether a Changes cache file is attached

Returns: boolean - Returns true if the Changes cache file is attached to the iModel. false otherwise

🔗queryChangeSummary StaticPublic

queryChangeSummary(iModel: IModelDb, changeSummaryId: Id64): ChangeSummary

Queries the ChangeSummary for the specified change summary id

throws IModelError If change summary does not exist for the specified id, or if the change cache file hasn't been attached, or in case of other errors.

Parameter Type Description
iModel IModelDb iModel
changeSummaryId Id64 ECInstanceId of the ChangeSummary (see ECDbChange.ChangeSummary ECClass)

Returns: ChangeSummary - Returns the requested ChangeSummary object

🔗queryInstanceChange StaticPublic

queryInstanceChange(iModel: IModelDb, instanceChangeId: Id64): InstanceChange

Queries the InstanceChange for the specified instance change id

throws IModelError if instance change does not exist for the specified id, or if the change cache file hasn't been attached, or in case of other errors.

Parameter Type Description
iModel IModelDb iModel
instanceChangeId Id64 ECInstanceId of the InstanceChange (see ECDbChange.InstanceChange ECClass)

Returns: InstanceChange - Returns the requested InstanceChange object
