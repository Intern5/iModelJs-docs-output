    🔗BriefcaseManager Class

    Utility to manage briefcases Folder structure for cached imodels: /assets/imodels/ => cachePath (can be specified) iModelId1/ => iModelPath csets/ => csetPath csetId1.cs csetid2.cs ... readOnly/ 0/IModelName.bim 1/IModelName.bim ... readWrite/ briefcaseId1/IModelName.bim briefcaseId2/IModelName.bim ... iModelId2/ ...
