    🔗IModelDbElements Class

    The collection of elements in an IModelDb.

    🔗Methods

    Name Description
    createElement ( elProps: ElementProps ): Element Public Create a new instance of an element.
    deleteElement ( id: Id64 ): void Public Delete an element from this iModel.
    getElement ( elementId: Id64Props | Guid | Code | ElementLoadProps ): Element Public Get an element by Id, FederationGuid, or Code
    getElementJson ( elementIdArg: string ): any Public Read element data from iModel as a json string
    getElementProps ( elementId: Id64Props | Guid | Code | ElementLoadProps ): ElementProps Public Get properties of an Element by Id, FederationGuid, or Code
    getMultiAspects ( elementId: Id64, aspectClassName: string ): ElementMultiAspect[] Public Get the ElementMultiAspect instances (by class name) that are related to the specified element.
    getRootSubject ( ): Subject Public Get the root subject element.
    getUniqueAspect ( elementId: Id64, aspectClassName: string ): ElementUniqueAspect Public Get an ElementUniqueAspect instance (by class name) that is related to the specified element.
    insertElement ( elProps: ElementProps ): Id64 Public Insert a new element into the iModel.
    queryChildren ( elementId: Id64 ): Id64[] Public Query for the child elements of the specified element.
    queryElementIdByCode ( code: Code ): Id64 | undefined Public Query for the DgnElementId of the element that has the specified code.
    updateElement ( props: ElementProps ): void Public Update some properties of an existing element.
