    IModelDb Class

    Represents a physical copy (a briefcase) of an iModel that can be accessed as a file on the local computer.

    IModelDb raises a set of events to allow apps and subsystems to track IModelDb object life cycle, including IModelDb.onOpen and IModelDb.onOpened.

    see learning about IModelDb
