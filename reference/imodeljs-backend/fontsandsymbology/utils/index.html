    🔗Utils Class

    Helper methods for creating and querying line styles

    🔗Methods

    Name Description
    createCompoundComponent ( iModel: IModelDb, props: CompoundProps ): StyleProps | undefined StaticPublic Create a file property for a new compound component.
    createPointSymbolComponent ( iModel: IModelDb, props: PointSymbolProps ): StyleProps | undefined StaticPublic Create a file property for a new point symbol component.
    createRasterComponent ( iModel: IModelDb, props: RasterImageProps, image: ArrayBuffer ): StyleProps | undefined StaticPublic Create a file property for a new raster image component.
    createStrokePatternComponent ( iModel: IModelDb, props: StrokePatternProps ): StyleProps | undefined StaticPublic Create a file property for a new stroke pattern component.
    createStrokePointComponent ( iModel: IModelDb, props: StrokePointProps ): StyleProps | undefined StaticPublic Create a file property for a new stroke point component.
    createStyle ( imodel: IModelDb, scopeModelId: Id64, name: string, props: StyleProps ): Id64 StaticPublic Insert a new line style with the supplied name.
    getOrCreateContinuousStyle ( imodel: IModelDb, scopeModelId: Id64, width: undefined | number ): Id64 StaticPublic Query for a continuous line style that can be used to create curves with physical width instead of weight in pixels and create one if it does not already exist.
    getOrCreateLinePixelsStyle ( imodel: IModelDb, scopeModelId: Id64, linePixels: LinePixels ): Id64 StaticPublic Query for a line style using the supplied LinePixels value (Code1-Code7) and create one if it does not already exist.
    queryStyle ( imodel: IModelDb, scopeModelId: Id64, name: string ): Id64 | undefined StaticPublic Query for an existing line style with the supplied name.
