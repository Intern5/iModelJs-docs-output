🔗GeometryStreamBuilder

GeometryStreamBuilder is a helper class for populating the GeometryStreamEntryProps array needed to create a GeometricElement or GeometryPart

🔗Methods

🔗appendGeometryParamsChange Public

appendGeometryParamsChange(geomParams: GeometryParams): boolean

Change GeometryParams for subsequent geometry. It is not valid to change the sub-category when defining a GeometryPart. GeometryParts inherit the symbology of their instance for anything not explicitly overridden.

Parameter Type Description
geomParams GeometryParams

Returns: boolean

🔗appendGeometryPart2d Public

appendGeometryPart2d(partId: Id64, instanceOrigin: Point2d, instanceRotation: Angle, instanceScale: undefined|number): boolean

Append a GeometryPart instance with relative position, orientation, and scale to a GeometryStreamEntryProps array for creating a GeometricElement2d. Not valid when defining a GeometryPart as nested GeometryParts are not allowed.

Parameter Type Description
partId Id64
instanceOrigin Point2d
instanceRotation Angle
instanceScale undefined|number

Returns: boolean

🔗appendGeometryPart3d Public

appendGeometryPart3d(partId: Id64, instanceOrigin: Point3d, instanceRotation: YawPitchRollAngles, instanceScale: undefined|number): boolean

Append a GeometryPart instance with relative position, orientation, and scale to a GeometryStreamEntryProps array for creating a GeometricElement3d. Not valid when defining a GeometryPart as nested GeometryParts are not allowed.

Parameter Type Description
partId Id64
instanceOrigin Point3d
instanceRotation YawPitchRollAngles
instanceScale undefined|number

Returns: boolean

🔗appendGeometryQuery Public

appendGeometryQuery(geometry: GeometryQuery): boolean

Append a GeometryQuery supplied in either local or world coordinates to the GeometryStreamProps array

Parameter Type Description
geometry GeometryQuery

Returns: boolean

🔗appendSubCategoryChange Public

appendSubCategoryChange(subCategoryId: Id64): boolean

Change sub-category or reset to sub-category appearance for subsequent geometry. An invalid sub-category id can be supplied to force a reset to the current sub-category appearance. It is not valid to change the sub-category when defining a GeometryPart. GeometryParts inherit the symbology of their instance for anything not explicitly overridden.

Parameter Type Description
subCategoryId Id64

Returns: boolean

🔗appendTextString Public

appendTextString(textString: TextString): boolean

Append a TextString supplied in either local or world coordinates to the GeometryStreamProps array

Parameter Type Description
textString TextString

Returns: boolean

🔗setLocalToWorld Public

setLocalToWorld(localToWorld: Transform): void

Supply optional local to world transform. Used to transform world coordinate input relative to element placement. For a GeometricElement's placement to be meaningful, world coordinate geometry should never be appended to an element with an identity placement. Can be called with undefined or identity transform to start appending geometry supplied in local coordinates again.

Parameter Type Description
localToWorld Transform

Returns: void

🔗setLocalToWorld2d Public

setLocalToWorld2d(origin: Point2d, angle: Angle = Angle.createDegrees(0.0)): void

Supply local to world transform from Point2d and Angle

Parameter Type Description
origin Point2d
angle Angle

Returns: void

🔗setLocalToWorld3d Public

setLocalToWorld3d(origin: Point3d, angles: YawPitchRollAngles = YawPitchRollAngles.createDegrees(0.0, 0.0, 0.0)): void

Supply local to world transform from Point3d and YawPitchRollAngles

Parameter Type Description

angles YawPitchRollAngles
