    🔗FeatureTable Class

    Defines a look-up table for Features within a batched RenderGraphic. Consecutive 32-bit indices are assigned to each unique Feature. Primitives within the RenderGraphic can use per-vertex indices to specify the distribution of Features within the primitive. A FeatureTable can be shared amongst multiple primitives within a single RenderGraphic, and amongst multiple sub-Graphics of a RenderGraphic.
