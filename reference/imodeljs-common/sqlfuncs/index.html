    SqlFuncs

    Functions that you can call in ECSQL statements. These are not TypeScript functions -- you can only call them in ECSQL statements. For more information: Spatial Queries

    🔗Global Functions

    Title Description
    iModel_angles Construct a YawPitchRollAngles object from 3 values
    iModel_angles_maxdiff Return the maximum absolute difference among the angles in degrees.
    iModel_angles_value Get a member of a iModel_angles object
    iModel_bbox Create a bounding box from 6 values
    iModel_bbox_areaxy Compute the depth times the width of a bounding box
    iModel_bbox_depth Compute the "depth" of a bounding box
    iModel_bbox_height Compute the "height" of a bounding box
    iModel_bbox_overlaps Determine if the areas enclosed by two 3-D bounding boxes overlap
    iModel_bbox_volume Compute the volume of the bounding box
    iModel_bbox_width Compute the "width" of a bounding box
    iModel_placement_aabb Get the axis-aligned bounding box from a placement
    iModel_placement_angles Get the placement angles
    iModel_placement_eabb Get the element-aligned bounding box from a placement
    iModel_placement_origin Get the placement origin
    iModel_point_value Get a member of a iModel_Point object.
