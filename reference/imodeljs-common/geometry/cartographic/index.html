    🔗Cartographic Class

    A position defined by longitude, latitude, and height above the WSG84 ellipsoid .

    🔗Methods

    Name Description

    equals ( right: Cartographic ): boolean Public return true if this Cartographic is the same as right
    equalsEpsilon ( right: Cartographic, epsilon: number ): boolean Public Compares this Cartographic component-wise and returns true if they are within the provided epsilon,
    toEcef ( result: Point3d ): Point3d Public Returns an ECEF point from a Cartographic point
    toString ( ): string Public Creates a string representing this cartographic in the format '(longitude, latitude, height)'.

    fromDegrees ( longitude: number, latitude: number, height: number, result: Cartographic ): Cartographic StaticPublic Creates a new Cartographic instance from longitude and latitude specified in degrees.
    fromEcef ( cartesian: Point3d, result: Cartographic ): Cartographic | undefined StaticPublic Creates a new Cartographic instance from an ECEF Cartesian position.
    fromRadians ( longitude: number, latitude: number, height: number = 0, result: Cartographic ): Cartographic StaticPublic Creates a new Cartographic instance from longitude and latitude specified in radians.
