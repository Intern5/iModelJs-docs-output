    Rendering

    Classes for rendering geometry in views.

    Classes

    Name Description
    Camera The current position (eyepoint), lens angle, and focus distance of a camera.
    ColorIndex
    EdgeArgs
    Feature Describes a "feature" within a batched RenderGraphic.
    FeatureIndex
    FeatureTable Defines a look-up table for Features within a batched RenderGraphic.
    FrustumPlanes Represents a frustum as 6 planes and provides containment and intersection testing
    GeometryParams Geometry display properties used to override or augment the SubCategory Appearance.
    GraphicParams The "cooked" material and symbology for a RenderGraphic.
    ImageBuffer Uncompressed bitmap image data
    ImageSource Image data encoded and compressed in either Jpeg or Png format.
    MeshEdge
    MeshEdges
    MeshPolyline
    MeshPolylineList
    NonUniformColor
    OctEncodedNormal
    OctEncodedNormalPair

    PolylineData
    PolylineEdgeArgs
    PolylineFlags Flags describing a polyline.
    QParams2d Parameters used for quantization of 2d points.
    QParams3d Parameters used for quantization of 3d points.
    QPoint2d Represents a quantized 2d point as an (x, y) pair in the integer range [0, 0xffff].
    QPoint2dList A list of 2d points all quantized to the same range.
    QPoint3d Represents a quantized 3d point as an (x, y, z) tripliet in the integer range [0, 0xffff].
    QPoint3dList A list of 3d points all quantized to the same range.
    RenderMaterial A Material for rendering
    RenderMaterial.Params
    RenderTexture A Texture for rendering

    SceneLights A list of Lights, plus the f-stop setting for the camera
