    Quadrature Class

    Support class for quadrature -- approximate integrals by summing weighted function values. These are filled with x and weight for quadrature between xA and xB

    Since quadrature is done in side tight loops, these methods are structured for minimum object allocation. For methods names setupGauss (where N is a smallish integer), the CALLER creates arrays xMapped and wMapped The method installs particular x and weight values.

    Methods

    Name Description
    mapWeights(xA: number, h: number, xRef: Float64Array, wRef: Float64Array, xMapped: Float64Array, wMapped: Float64Array): number Static Given points and weights in a reference interval (usually 0 to 1):

    map each xRef[i] to xA + h xRef[i];
    scale each weight wRef[i] to h wRef[i]
    * all arrays are assumed to have xRef.length entries.
    setupGauss2(xA: number, xB: number, xMapped: Float64Array, wMapped: Float64Array): number Static
    setupGauss3(xA: number, xB: number, xMapped: Float64Array, wMapped: Float64Array): number Static
    setupGauss4(xA: number, xB: number, xMapped: Float64Array, wMapped: Float64Array): number Static
    setupGauss5(xA: number, xB: number, xMapped: Float64Array, wMapped: Float64Array): number Static Caller allocates and passes Float6dArray of length
    These are filled with x and weight for quadrature between xA and xB
    sum1(xx: Float64Array, ww: Float64Array, n: number, f: (x: number) => number): number Static Sum function values with given weghts and x values.
