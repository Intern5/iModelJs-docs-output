    🔗ClipPlaneSet Class

    A collection of ConvexClipPlaneSets.

    • A point is "in" the clip plane set if it is "in" one or more of the ConvexClipPlaneSet
    • Hence the boolean logic is that the ClipPlaneSet is a UNION of its constituents.
