    OrderedRotationAngles Class

    OrderedRotationAngles represents a non-trivial rotation using three simple axis rotation angles, and an order in which to apply them.

    Methods

    Name Description
    toMatrix3d(result?: Matrix3d): Matrix3d Create a 3x3 rotational matrix from this OrderedRotationAngles.
    createAngles(xRotation: Angle, yRotation: Angle, zRotation: Angle, order: AxisOrder, result?: OrderedRotationAngles): OrderedRotationAngles Static Create an OrderedRotationAngles from three angles and an ordering in which to apply them when rotating.
    createDegrees(xDegrees: number, yDegrees: number, zDegrees: number, order: AxisOrder, result?: OrderedRotationAngles): OrderedRotationAngles Static Create an OrderedRotationAngles from three angles (in degrees) and an ordering in which to apply them when rotating.
    createFromMatrix3d(matrix: Matrix3d, order: AxisOrder, result?: OrderedRotationAngles): OrderedRotationAngles Static Create an OrderedRotationAngles from a 3x3 rotational matrix, given the ordering of axis rotations that the matrix derives from.
    createRadians(xRadians: number, yRadians: number, zRadians: number, order: AxisOrder, result?: OrderedRotationAngles): OrderedRotationAngles Static Create an OrderedRotationAngles from three angles (in radians) and an ordering in which to apply them when rotating.
