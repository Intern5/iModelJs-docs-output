🔗PolygonOps - Class

Static class for operations that treat an array of points as a polygon (with area!)

🔗addSecondMomentAreaProducts StaticPublic

addSecondMomentAreaProducts(points: IndexedXYZCollection, origin: Point3d, moments: Matrix4d): void

Return the matrix of area products of a polygon with respect to an origin. The polygon is assumed to be planar and non-self-intersecting.

Parameter Type Description
points IndexedXYZCollection
origin Point3d
moments Matrix4d

Returns: void

sumTriangleAreas(points: Point3d[]): number

Sum areas of triangles from points[0] to each far edge.

  • Sum the areas(absolute, without regard to orientation) all these triangles.
