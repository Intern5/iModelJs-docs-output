🔗Ray3d

A Ray3d contains

  • an origin point.

  • a direction vector. The vector is NOT required to be normalized.

    • an optional weight (number).

spacePoint Point3d

Parameter Type Description
fraction number

Returns: Point3d - Return a point at fractional position along the ray.
