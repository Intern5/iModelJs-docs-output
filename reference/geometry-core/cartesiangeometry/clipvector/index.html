🔗classifyPointContainment Public

classifyPointContainment(points: Point3d[], ignoreMasks: boolean = false): ClipPlaneContainment

Determines whether the given points fall inside or outside this set of ClipShapes. If any set is defined by masking planes, checks the mask planes only, provided that ignoreMasks is false. Otherwise, checks the _clipplanes member.

Parameter Type Description
points Point3d[]
ignoreMasks boolean

Returns: ClipPlaneContainment

🔗classifyRangeContainment Public

classifyRangeContainment(range: Range3d, ignoreMasks: boolean): ClipPlaneContainment

Determines whether a 3D range lies inside or outside this set of ClipShapes. If any set is defined by masking planes, checks the mask planes only, provided that ignoreMasks is false. Otherwise, checks the _clipplanes member.

Parameter Type Description
range Range3d
ignoreMasks boolean

Returns: ClipPlaneContainment
