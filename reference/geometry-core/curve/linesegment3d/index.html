🔗LineSegment3d

A LineSegment3d is:

  • A 3d line segment represented by

startPoint endPoint parameterized with fraction 0 at the start and fraction 1 at the end, i.e. either of these equivalent forms:

X(f) = startPoint + f * (endPoint - startPoint) X(f) = (1-f)*startPoint + f * endPoint
