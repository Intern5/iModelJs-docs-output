    🔗IModelJson Namespace

    🔗Class

    Name Description
    🔗Reader parser servoces for "iModelJson" schema
    🔗Writer

    🔗Interface

    Name Description
    🔗ArcByVectorProps Interface for Arc3d value defined by center, vectorX, vectorY and sweepStartEnd.
    🔗AxesProps * There are multiple ways to specify an orientation
    🔗BSplineSurfaceProps Right side (content) properties for {bsurf: BSplineSurfaceProps}
    🔗BcurveProps Interface for bspline curve (aka bcurve)
    🔗BoxProps Interface for Box (or frustum with all rectangular sections parallel to primary xy section)
    🔗ConeProps Interface for Cone value defined by centers, radii, and (optional) vectors for circular section planes.
    🔗CurveCollectionProps Interface for a collection of curves, eg.
    🔗CurvePrimitiveProps
    🔗CylinderProps Interface for cylinder defined by a radius and axis start and end centers.
    🔗GeometryProps
    🔗IndexedMeshProps Interface for an indexed mesh.
    🔗LinearSweepProps Interface for a linear sweep of a base curve or region.
    🔗PlanarRegionProps Interface for a collection of curves that bound a planar region
    🔗PointProps
    🔗RotationalSweepProps Interface for a rotational sweep of a base curve or region around an axis.
    🔗RuledSweepProps Interface for a surface with ruled sweeps between corresponding curves on successvie contours
    🔗SolidPrimitiveProps Interface for solid primitives: box, sphere, cylinder, cone, torusPipe, linear sweep, rotational sweep, ruled sweep.
    🔗SphereProps Interface for Sphere (with optionally different radius to pole versus equator)
    🔗TorusPipeProps Interface for TorusPipe data
    🔗TransitionSpiralProps Interface for spiral
