🔗overwriteWithScaledCombinations Public

overwriteWithScaledCombinations(i0: number, i1: number, f0: number, f1: number): void

  • For each index i0 <= i < i1 overwrite data[i+1] by `f0data[i]+f1data[i+1]
  • This is the essential step of a bezier polynomial subdivision step
Parameter Type Description
i0 number first index to update
i1 number one beyond last index to update.
f0 number left scale
f1 number right scale

🔗resize Public

resize(newLength: number, padValue: number = 0): void

  • If newLength is less than current (active) length, just set (active) length.
  • If newLength is greater, ensureCapacity (newSize) and pad with padValue up to newSize;
Parameter Type Description
newLength number new data count
padValue number value to use for padding if the length increases.

Returns: void

restrictToInterval(a: number, b: number): void
