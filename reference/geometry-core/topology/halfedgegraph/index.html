🔗HalfEdgeGraph

A HalfEdgeGraph has:

  • An array of (pointers to ) HalfEdge objects.

🔗Constructors

Returns: HalfEdgeGraph

🔗addEdgeXY Public

addEdgeXY(x0: number, y0: number, x1: number, y1: number): HalfEdge

create two nodes of a new edge.

Parameter Type Description
x0 number
y0 number
x1 number
y1 number

Returns: HalfEdge - Return one of the two nodes, which the caller may consider as the start of the edge.

🔗announceFaceLoops Public

announceFaceLoops(announceFace: GraphNodeFunction): void

  • Visit each facet of the graph once.
  • Call the announceFace function
  • continue search if announceFace(graph, node) returns true
  • terminate search if announceface (graph, node) returns false
Parameter Type Description
announceFace GraphNodeFunction

Returns: void

🔗announceVertexLoops Public

announceVertexLoops(announceVertex: GraphNodeFunction): void

  • Visit each vertex loop of the graph once.
  • Call the announceVertex function
  • continue search if announceFace(graph, node) returns true
  • terminate search if announceface (graph, node) returns false
Parameter Type Description
announceVertex GraphNodeFunction

Returns: void

🔗collectFaceLoops Public

collectFaceLoops(): HalfEdge[]

Returns: HalfEdge[] - Returns an array of nodes, where each node represents a starting point of a face loop.

🔗collectSegments Public

collectSegments(): LineSegment3d[]

Return an array LineSegment3d.

  • The array has one segment per edge
  • The coordinates are taken from a node and its face successor.
  • On each edge, the line segment start at the HalfEdge with lower id than its edgeMate.

Returns: LineSegment3d[]

🔗collectVertexLoops Public

collectVertexLoops(): HalfEdge[]

Returns: HalfEdge[] - Returns an array of nodes, where each node represents a starting point of a vertex loop.

🔗createEdgeXYZXYZ Public

createEdgeXYZXYZ(xA: number = 0, yA: number = 0, zA: number = 0, iA: number = 0, xB: number = 0, yB: number = 0, zB: number = 0, iB: number = 0): HalfEdge

  • Create 2 half edges forming 2 vertices, 1 edge, and 1 face
  • The two edges are joined as edgeMate pair.
  • The two edges are a 2-half-edge face loop in both the faceSuccessor and facePredecessor directions.
  • The two edges are added to the graph's HalfEdge set
Parameter Type Description
xA number
yA number
zA number
iA number
xB number
yB number
zB number
iB number

Returns: HalfEdge - Return pointer to the first half edge created.

🔗decommission Public

decommission(): void

This is a destructor-like action that elminates all interconnection among the graph's nodes. After this is called the graph is unusable.

Returns: void

🔗splitEdge Public

splitEdge(base: undefined|HalfEdge, xA: number = 0, yA: number = 0, zA: number = 0, iA: number = 0): HalfEdge

  • Insert a vertex in the edge begining at base.
  • this creates two half edges.
  • The base of the new edge is 'after' the (possibly undefined) start node in its face loop.
  • The existing mate retains its base xyzi properties but is no longer the mate of base.
  • The base and existing mate each become mates with a new half edge.
Parameter Type Description
base undefined|HalfEdge
xA number
yA number
zA number
iA number

Returns: HalfEdge - Returns the reference to the half edge created.
