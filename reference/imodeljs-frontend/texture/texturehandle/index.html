    🔗TextureHandle Class

    Encapsulates a handle to a WebGLTexture object created based on desired parameters.

    🔗Methods

    Name Description
    bind ( texUnit: TextureUnit ): boolean Public Binds texture handle (if available) associated with an instantiation of this class to specified texture unit.
    bindSampler ( uniform: UniformHandle, unit: TextureUnit ): void Public Binds this texture to a uniform sampler2D
    dispose ( ): void Public
    getHandle ( ): WebGLTexture | undefined Public Retrieves actual WebGLTexture object associated with this texture.
    bindSampler ( uniform: UniformHandle, tex: WebGLTexture, unit: TextureUnit ): void StaticPublic Binds the specified texture to a uniform sampler2D
    bindTexture ( texUnit: TextureUnit, glTex: WebGLTexture | undefined ): void StaticPublic Binds specified texture handle to specified texture unit.
    createForAttachment ( width: number, height: number, format: Format, dataType: DataType ): undefined | TextureHandle StaticPublic Creates a texture for a framebuffer attachment (no data specified).
    createForData ( width: number, height: number, data: Uint8Array, wantPreserveData: boolean = false ): undefined | TextureHandle StaticPublic Creates a texture for storing data accessed by shaders.
    createForImage ( width: number, height: number, imageBytes: Uint8Array, imageFormat: ImageSourceFormat, isTranslucent: boolean, loadCallback: TextureLoadCallback, useMipMaps: boolean = true, isGlyph: boolean = false, isTileSection: boolean = false, wantPreserveData: boolean = false ): undefined | TextureHandle StaticPublic Creates a texture for a jpeg or png image.
