    🔗TextureHandle Class

    Encapsulates a handle to a WebGLTexture object created based on desired parameters.

    bindSampler ( uniform: UniformHandle, unit: TextureUnit ): void Public Binds this texture to a uniform sampler2D
    dispose ( ): void Public
    getHandle ( ): WebGLTexture|undefined Public Retrieves actual WebGLTexture object associated with this texture.
