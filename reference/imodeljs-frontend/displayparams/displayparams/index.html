    🔗DisplayParams Class

    This class is used to determine if things can be batched together for display.

    🔗Methods

    Name Description
    equals ( rhs: DisplayParams, purpose: ComparePurpose = DisplayParams.ComparePurpose.Strict ): boolean Public Determines if the properties of this DisplayParams object are equal to those of another DisplayParams object.
    createForLinear ( gf: GraphicParams ): DisplayParams StaticPublic Creates a DisplayParams object that describes linear geometry based on the specified GraphicParams.
    createForMesh ( gf: GraphicParams ): DisplayParams StaticPublic Creates a DisplayParams object that describes mesh geometry based on the specified GraphicParams.
    createForText ( gf: GraphicParams ): DisplayParams StaticPublic Creates a DisplayParams object that describes text geometry based on the specified GraphicParams.
    createForType ( type: Type, gf: GraphicParams ): DisplayParams StaticPublic Creates a DisplayParams object for a particular type (mesh, linear, text) based on the specified GraphicParams.
