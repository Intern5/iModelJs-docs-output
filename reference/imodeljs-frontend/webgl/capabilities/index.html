    Capabilities Class

    Describes the rendering capabilities of the host system.

    Methods

    Name Description
    init(gl: WebGLRenderingContext): boolean Initializes the capabilities based on a GL context.
    queryExtensionObject<T>(ext: string): T | undefined Queries an extension object if available.
    create(gl: WebGLRenderingContext): Capabilities | undefined Static

    Properties

    Name Type Description
    maxColorAttachments Accessor ReadOnly number
    maxDepthType Accessor ReadOnly DepthType
    maxDrawBuffers Accessor ReadOnly number
    maxFragTextureUnits Accessor ReadOnly number
    maxFragUniformVectors Accessor ReadOnly number
    maxRenderType Accessor ReadOnly RenderType
    maxTextureSize Accessor ReadOnly number
    maxVaryingVectors Accessor ReadOnly number
    maxVertAttribs Accessor ReadOnly number
    maxVertTextureUnits Accessor ReadOnly number
    maxVertUniformVectors Accessor ReadOnly number
    supports32BitElementIndex Accessor ReadOnly boolean
    supportsDrawBuffers Accessor ReadOnly boolean
    supportsMRTPickShaders Accessor ReadOnly boolean
    supportsMRTTransparency Accessor ReadOnly boolean
    supportsNonPowerOf2Textures Accessor ReadOnly boolean These getters check for existence of extension objects to determine availability of features. In WebGL2, could just return true for some.
    supportsShaderTextureLOD Accessor ReadOnly boolean
    supportsTextureFloat Accessor ReadOnly boolean
