    🔗IModelConnection Class

    A connection to an iModel database hosted on the backend.

    🔗Extends

    • IModel

    🔗Implements

    • IModelProps

    🔗Methods

    Name Description
    attachChangeCache ( ): Promise<void> Public Attaches the Changes Cache File to this iModel if it hasn't been attached yet.
    close ( accessToken: AccessToken ): Promise<void> Public Close this IModelConnection
    closeStandalone ( ): Promise<void> Public Close this standalone IModelConnection
    executeQuery ( ecsql: string, bindings?: any[] | any ): Promise<any[]> Public Execute a query against the iModel.
    isChangeCacheAttached ( ): Promise<boolean> Public Determines whether the Changes Cache File is attached to this iModel or not.
    isReadonly ( ): boolean Check if this iModel has been opened read-only or not.
    loadFontMap ( ): Promise<FontMap> Public Load the FontMap for this IModelConnection.
    queryEntityIds ( params: EntityQueryParams ): Promise<Id64Set> Public query for a set of ids that satisfy the supplied query params
    saveChanges ( description?: undefined | string ): Promise<void> Public Commit pending changes to this iModel
    toJSON ( ): any
    updateProjectExtents ( newExtents: AxisAlignedBox3d ): Promise<void> Public Update the project extents of this iModel.
    getDefaultSubCategoryId ( categoryId: Id64 ): Id64 Static get the default subCategoryId for the supplied categoryId
    open ( accessToken: AccessToken, contextId: string, iModelId: string, openMode: OpenMode = OpenMode.Readonly, version: IModelVersion = IModelVersion.latest() ): Promise<IModelConnection> StaticPublic Open an IModelConnection to an iModel
    openStandalone ( fileName: string, openMode: OpenMode = OpenMode.Readonly ): Promise<IModelConnection> StaticPublic Open an IModelConnection to a standalone iModel (not managed by iModelHub) from a file name that is resolved by the backend.
