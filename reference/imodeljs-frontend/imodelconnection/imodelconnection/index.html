    IModelConnection Class

    A connection to an iModel database hosted on the backend.

    Extends

    • IModel

    Implements

    • IModelProps

    Methods

    Name Description
    attachChangeCache(): Promise<void> Attaches the Change Cache file to this iModel if it hasn't been attached yet.
    close(accessToken: AccessToken): Promise<void> Close this IModelConnection
    closeStandalone(): Promise<void> Close this standalone IModelConnection
    detachChangeCache(): Promise<void> Detaches the Change Cache file to this iModel if it had been attached before.
    executeQuery(ecsql: string, bindings?: any[] | any): Promise<any[]> Execute a query against the iModel.
    isChangeCacheAttached(): Promise<boolean> Determines whether the Change Cache file is attached to this iModel or not.
    isReadonly(): boolean Check if this iModel has been opened read-only or not.
    loadFontMap(): Promise<FontMap> Load the FontMap for this IModelConnection.
    queryEntityIds(params: EntityQueryParams): Promise<Id64Set> query for a set of ids that satisfy the supplied query params
    saveChanges(description?: string): Promise<void> Commit pending changes to this iModel
    updateProjectExtents(newExtents: AxisAlignedBox3d): Promise<void> Update the project extents of this iModel.
    open(accessToken: AccessToken, contextId: string, iModelId: string, openMode: OpenMode = OpenMode.Readonly, version: IModelVersion = IModelVersion.latest()): Promise<IModelConnection> Static Open an IModelConnection to an iModel
    openStandalone(fileName: string, openMode: OpenMode = OpenMode.Readonly): Promise<IModelConnection> Static Open an IModelConnection to a standalone iModel (not managed by iModelHub) from a file name that is resolved by the backend.
