🔗executeQuery Public

executeQuery(ecsql: string, bindings: any[]|any): Promise

Execute a query against the iModel.

throws IModelError if the ECSQL is invalid

Parameter Type Description
ecsql string The ECSQL to execute
bindings any[]|any The values to bind to the parameters (if the ECSQL has any).

Pass an array if the parameters are positional. Pass an object of the values keyed on the parameter name for named parameters. The values in either the array or object must match the respective types of the parameters.

Supported types:

  • boolean
  • number for integral or double parameters
  • string for string parameters,
  • ECSqlTypedString for date time, blob, id, or guid parameters
  • Id64 for id parameters
  • Guid for guid parameters
  • NavigationBindingValue for navigation property parameters
  • XAndY for Point2d parameters
  • XYAndZ for Point3d parameters
  • Objects of primitives, objects or arrays of any of the above types when binding structs
  • Arrays of primitives or objects of any of the above when binding arrays

Returns: Promise - Returns the query result as an array of the resulting rows or an empty array if the query has returned no rows
