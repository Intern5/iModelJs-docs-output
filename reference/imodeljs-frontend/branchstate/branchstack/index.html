    🔗BranchStack Class

    Represents the current state of the scene graph. As the scene graph is traversed, branch states are pushed and popped. Pushing a branch state replaces the current view flags and multiplies the current transform with the branch's transform. Popping it inverts this operation. The state at the top of the stack applies to the rendering of all primitives. The stack does not store the scene graph itself.
