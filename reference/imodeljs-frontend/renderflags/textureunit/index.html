    🔗TextureUnit Enumeration

    Reserved texture units for specific sampler variables, to avoid conflicts between shader components which each have their own textures.

    🔗AnimationLUT

    Default value: WebGLRenderingContext.TEXTURE12

    🔗BRDFLUT

    Default value: WebGLRenderingContext.TEXTURE10

    🔗ClipMask

    Default value: Zero

    🔗DiffuseMap

    Default value: WebGLRenderingContext.TEXTURE8

    🔗ElementId

    Default value: Three

    🔗EnvironmentMap

    Default value: WebGLRenderingContext.TEXTURE11

    🔗FeatureSymbology

    Default value: One

    🔗Five

    Default value: WebGLRenderingContext.TEXTURE5

    🔗Four

    Default value: WebGLRenderingContext.TEXTURE4

    🔗IBLSpecular

    Default value: WebGLRenderingContext.TEXTURE9

    🔗LineCode

    Default value: Two

    🔗One

    Default value: WebGLRenderingContext.TEXTURE1

    🔗PickDepthAndOrder

    Default value: Six

    🔗PickElementId0

    Default value: Four

    🔗PickElementId1

    Default value: Five

    🔗Seven

    Default value: WebGLRenderingContext.TEXTURE7

    🔗Six

    Default value: WebGLRenderingContext.TEXTURE6

    🔗SurfaceTexture

    Default value: Two

    🔗Three

    Default value: WebGLRenderingContext.TEXTURE3

    🔗Two

    Default value: WebGLRenderingContext.TEXTURE2

    🔗VertexLUT

    Default value: Seven

    🔗Zero

    Default value: WebGLRenderingContext.TEXTURE0
