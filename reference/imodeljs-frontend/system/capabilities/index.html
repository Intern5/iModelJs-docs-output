    🔗Capabilities Class

    Describes the rendering capabilities of the host system.

    🔗Methods

    Name Description
    init ( gl: WebGLRenderingContext ): boolean Public Initializes the capabilities based on a GL context.
    queryExtensionObject<T> ( ext: string ): T | undefined Public Queries an extension object if available.
    create ( gl: WebGLRenderingContext ): Capabilities | undefined StaticPublic
