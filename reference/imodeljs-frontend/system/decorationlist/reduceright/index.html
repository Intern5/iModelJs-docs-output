    🔗reduceRight Static

    Calls the specified callback function for all the elements in an array, in descending order. The return value of the callback function is the accumulated result, and is provided as an argument in the next call to the callback function.

    reduceRight(callbackfn: object): Decoration

    Parameter Type Description
    callbackfn object A function that accepts up to four arguments. The reduceRight method calls the callbackfn function one time for each element in the array.

    Returns: Decoration

    reduceRight(callbackfn: object, initialValue: Decoration): Decoration

    Parameter Type Description
    callbackfn object
    initialValue Decoration

    Returns: DecorationCalls the specified callback function for all the elements in an array, in descending order. The return value of the callback function is the accumulated result, and is provided as an argument in the next call to the callback function.

    reduceRight<U>(callbackfn: object, initialValue: U): U

    Parameter Type Description
    callbackfn object A function that accepts up to four arguments. The reduceRight method calls the callbackfn function one time for each element in the array.
    initialValue U If initialValue is specified, it is used as the initial value to start the accumulation. The first call to the callbackfn function provides this value as an argument instead of an array value.

    Returns: U
