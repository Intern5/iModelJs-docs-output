    🔗SnapMode Enumeration

    🔗AllConstraint

    Default value: (Tangency &#124; TangentPoint &#124; Perpendicular &#124; PerpendicularPoint &#124; Parallel &#124; PointOn)

    🔗AllOrdinary

    Default value: (Nearest &#124; NearestKeypoint &#124; MidPoint &#124; Center &#124; Origin &#124; Bisector &#124; Intersection &#124; MultiSnaps)

    🔗Bisector

    Default value: 1 << 5

    🔗Center

    Default value: 1 << 3

    🔗First

    Default value: 0

    🔗Intersection

    Default value: 1 << 6

    🔗IntersectionCandidate

    Default value: (Intersection &#124; Nearest)

    🔗Invalid

    Default value: -1

    🔗MidPoint

    Default value: 1 << 2

    🔗Multi1

    Default value: 1 << 14

    🔗Multi2

    Default value: 1 << 15

    🔗Multi3

    Default value: 1 << 12

    🔗MultiSnaps

    Default value: (Multi1 &#124; Multi2 &#124; Multi3)

    🔗Nearest

    Default value: 1

    🔗NearestKeypoint

    Default value: 1 << 1

    🔗None

    Default value: 0

    🔗NumSnapModes

    Default value: 16

    🔗Origin

    Default value: 1 << 4

    🔗Parallel

    Default value: 1 << 11

    🔗Perpendicular

    Default value: 1 << 9

    🔗PerpendicularPoint

    Default value: 1 << 10

    🔗PointOn

    Default value: 1 << 13

    🔗Tangency

    Default value: 1 << 7

    🔗TangentPoint

    Default value: 1 << 8
