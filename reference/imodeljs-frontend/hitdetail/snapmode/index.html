    🔗SnapMode Enumeration

    🔗AllConstraint

    Default value: (Tangency <code>&#124;</code> TangentPoint <code>&#124;</code> Perpendicular <code>&#124;</code> PerpendicularPoint <code>&#124;</code> Parallel <code>&#124;</code> PointOn)

    🔗AllOrdinary

    Default value: (Nearest <code>&#124;</code> NearestKeypoint <code>&#124;</code> MidPoint <code>&#124;</code> Center <code>&#124;</code> Origin <code>&#124;</code> Bisector <code>&#124;</code> Intersection <code>&#124;</code> MultiSnaps)

    🔗Bisector

    Default value: 1 << 5

    🔗Center

    Default value: 1 << 3

    🔗First

    Default value: 0

    🔗Intersection

    Default value: 1 << 6

    🔗IntersectionCandidate

    Default value: (Intersection <code>&#124;</code> Nearest)

    🔗Invalid

    Default value: -1

    🔗MidPoint

    Default value: 1 << 2

    🔗Multi1

    Default value: 1 << 14

    🔗Multi2

    Default value: 1 << 15

    🔗Multi3

    Default value: 1 << 12

    🔗MultiSnaps

    Default value: (Multi1 <code>&#124;</code> Multi2 <code>&#124;</code> Multi3)

    🔗Nearest

    Default value: 1

    🔗NearestKeypoint

    Default value: 1 << 1

    🔗None

    Default value: 0

    🔗NumSnapModes

    Default value: 16

    🔗Origin

    Default value: 1 << 4

    🔗Parallel

    Default value: 1 << 11

    🔗Perpendicular

    Default value: 1 << 9

    🔗PerpendicularPoint

    Default value: 1 << 10

    🔗PointOn

    Default value: 1 << 13

    🔗Tangency

    Default value: 1 << 7

    🔗TangentPoint

    Default value: 1 << 8
