    🔗Viewport Class

    A Viewport maps a set of one or more Models to an output device. It holds a ViewState that defines the viewing parameters.

    🔗Accessors

    Name Type Access Description
    🔗auxCoordSystem AuxCoordSystemState readOnly
    🔗iModel IModelConnection readOnly
    🔗isActive boolean readOnly Determine whether this Viewport is currently active. Viewports become "active" after they have been initialized and connected to an Render::Target.
    🔗isGridOn boolean readOnly Determine whether the Grid display is currently enabled in this DgnViewport.
    🔗isRedoPossible boolean readOnly True if an redoable viewing operation exists on the stack
    🔗isUndoPossible boolean readOnly True if an undoable viewing operation exists on the stack
    🔗pixelsPerInch number readOnly
    🔗target RenderTarget readOnly
    🔗view ViewState readOnly the current ViewState controlling this Viewport
    🔗viewCmdTargetCenter Point3d | undefined readWrite
    🔗viewFlags ViewFlags readOnly
    🔗viewRect ViewRect readOnly get the rectangle of this Viewport in ViewCoordinates.
    🔗wantAntiAliasLines AntiAliasPref readOnly
    🔗wantAntiAliasText AntiAliasPref readOnly
