    🔗Classes

    Name Description
    🔗AuxCoordSystem2dState
    🔗AuxCoordSystem3dState
    🔗AuxCoordSystemSpatialState
    🔗AuxCoordSystemState
    🔗CategorySelectorState A list of Categories to be displayed in a view.
    🔗DecorateContext
    🔗DecorationAnimator A ViewportAnimator that animated decorations.
    🔗DepthRangeNpc the minimum and maximum values for the "depth" of a rectangle of screen space.
    🔗DisplayStyle2dState A DisplayStyle for 2d views
    🔗DisplayStyle3dState A DisplayStyle for 3d views
    🔗DisplayStyleState A DisplayStyle defines the parameters for 'styling' the contents of a View
    🔗DrawingViewState A view of a DrawingModel
    🔗DynamicsContext
    🔗Environment the skyBox, groundPlane, etc.
    🔗GroundPlane A circle drawn at a Z elevation, whose diameter is the the XY diagonal of the project extents
    🔗MarginPercent Margins for white space to be left around view volumes for ViewDefinition.lookAtVolume.
    🔗ModelSelectorState A set of ids of GeometricModels for a SpatialViewDefinition.
    🔗NullContext
    🔗OrthographicViewState Defines a spatial view that displays geometry on the image plane using a parallel orthographic projection.
    🔗RenderContext
    🔗SceneContext
    🔗SheetViewState A view of a SheetModel
    🔗SkyBox the SkyBox is a grid drawn in the background of spatial views to provide context.
    🔗SnapContext
    🔗SpatialViewState Defines a view of one or more SpatialModels.
    🔗SpecialElements Encapsulates logic for handling always/never drawn sets
    🔗Sprite Sprites are small raster images that are drawn "on top" of Viewports by a ViewDecoration.
    🔗SpriteLocation A Sprite Location.
    🔗SyncFlags viewport synchronization flags
    🔗ViewContext
    🔗ViewManager The ViewManager holds the list of opened views, plus the "selected view"
    🔗Viewport A Viewport maps a set of one or more Models to an output device.
    🔗ViewRect A rectangle in view coordinates.
    🔗ViewState The state of a ViewDefinition element.
    🔗ViewState2d Defines the state of a view of a single 2d model.
    🔗ViewState3d Defines the state of a view of 3d models.
