    🔗ViewManager Class

    The ViewManager holds the list of opened views, plus the "selected view"

    Name Description
    🔗addViewport(newVp: Viewport): void Public Add a new Viewport to the list of opened views and create an EventController for it.
    🔗beginDynamicsMode(): void Public
    🔗callDecorators(context: DecorateContext): void Public
    🔗clearInfoWindow(): void Public
    🔗doesHostHaveFocus(): boolean Public
    🔗dropViewport(vp: Viewport): BentleyStatus Public Remove a Viewport from the list of opened views.
    🔗endDynamicsMode(): void Public
    🔗forEachViewport(func: (vp: Viewport) => void): void Public
    🔗getFirstOpenView(): Viewport | undefined Public Get the first opened view.

    🔗validateViewportScenes(): void Public

    🔗Accessors

    Name Type Access Description
    🔗selectedView undefined | Viewport readWrite The "selected view" is the default for certain operations.

    🔗Properties

    Name Type Description
    🔗cursor BeCursor
    🔗inDynamicsMode boolean
    🔗onSelectedViewportChanged BeEvent<(old: Viewport | undefined, current: Viewport | undefined) => void> Called after the selected view changes.
    🔗onViewClose BeEvent<(vp: Viewport) => void> Called after a view is closed.
    🔗onViewOpen BeEvent<(vp: Viewport) => void> Called after a view is opened.
    🔗onViewResume BeEvent<(vp: Viewport) => void> Called after a suspended view is resumed.
    🔗onViewSuspend BeEvent<(vp: Viewport) => void> Called after a view is suspended.
