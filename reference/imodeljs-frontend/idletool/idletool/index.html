🔗IdleTool

The default "idle" tool. If no tool is active, or the active tool does not respond to a given event, input events are forwarded to the idle tool. The default idle tool converts middle mouse button events and touch gestures into view navigation operations like pan, zoom, rotate, and fit.

🔗getInfoString Public

getInfoString(hit: HitDetail, _delimiter: string): string

Invoked just before the locate tooltip is displayed to retrieve the info text. Allows the tool to override the default description.

note If you override this method, you may decide whether to call your superclass' implementation or not (it is not required). The default implementation shows hit description

Parameter Type Description
hit HitDetail The HitDetail whose info is needed.
_delimiter string Use this string to break lines of the description.

Returns: string - the string to describe the hit.
