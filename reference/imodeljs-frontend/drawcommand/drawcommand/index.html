    🔗DrawCommand Class

    Represents a command to be executed within a RenderPass. The most common command is to draw a primitive; others involve state changes such as pushing/popping transforms and symbology overrides, which require that commands be executed in order.

    🔗Accessors

    get branch(): Branch|undefined

    get featureIndexType(): FeatureIndexType

    get hasFeatureOverrides(): boolean

    get isPrimitiveCommand(): boolean

    get primitive(): Primitive|undefined

    get pushOrPop(): PushOrPop|undefined

    get renderOrder(): RenderOrder
