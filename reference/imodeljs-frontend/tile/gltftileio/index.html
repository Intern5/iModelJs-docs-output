    GltfTileIO Namespace

    Provides facilities for deserializing glTF tile data.

    Enumeration

    Name Description
    GltfTileIO.Constants
    GltfTileIO.DataType
    GltfTileIO.PrimitiveType
    GltfTileIO.Versions Known version of the glTF format.

    Class

    Name Description
    GltfTileIO.BufferData A chunk of binary data exposed as a typed array.
    GltfTileIO.BufferView A view of a chunk of a tile's binary data containing an array of elements of a specific data type.
    GltfTileIO.Header Header preceding glTF tile data.
    GltfTileIO.Reader Deserializes glTF tile data.
    GltfTileIO.ReaderProps Data required for creating a Reader capable of deserializing glTF tile data.

    Interface

    Name Description
    GltfTileIO.ReaderResult The result of Reader.read().

    Type alias

    Name Description
    GltfTileIO.DataBuffer
    GltfTileIO.IsCanceled A function that returns true if Reader.read() should abort because the tile data is no longer needed.
