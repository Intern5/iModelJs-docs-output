    Classes for selecting and drawing tiles in views.

    Name Description
    IModelTileLoader
    MissingNodes
    Tile
    Tile.DrawArgs Arguments used when selecting and drawing tiles.
    Tile.Params Parameters used to construct a Tile.
    TileLoader
    TileRequests
    TileTree
    TileTree.Params Parameters used to construct a TileTree

    Enumerations

    Name Description
    Tile.LoadStatus Describes the current status of a Tile.
    Tile.SelectParent Returned by Tile.selectTiles() to indicate whether a parent tile should be drawn in place of a child tile.
    Tile.Visibility Describes the visibility of a tile based on its size and a view frustum.

    PntsTileIO Deserializes an Pnts tile.
    TileIO Provides facilities for deserializing 3d tiles
