    ui-core

    Classes

    Name Description
    ContextMenu A context menu populated with ContextMenItem components.
    ContextMenuDivider Menu Divider for Context Menu.
    ContextMenuItem Menu Item class for use within a ContextMenu component.
    ContextSubMenu Submenu wrapper class for use within a ContextMenu component.
    Dialog Dialog component with optional resizing and dragging functionality
    Div Div React component is a wrapper for the HTML div element
    ExpansionToggle
    MessageBox Message Box React component.
    MessageContainer Message Container React component.
    RadialButton Button for use within RadialMenu
    RadialMenu A context menu arranged in a radial layout.
    SearchBox Input box with builtin icon right justified bounded by the SearchBox
    SplitButton SplitButton with a button on the left and a context menu on the right.
    Timer Notifies handler after a set interval.
    Tree Presentation React component for a Tree
    TreeBranch Presentation React component for a Tree branch
    TreeNode Presentation React component for a Tree node
    UiCore Entry point for static initialization required by various
    UiEvent IModelJs UI Event class, which is a subclass of BeEvent with argument type safety.
    ValidationTextbox Input box that validates text based on provided criteria.

    Name Description

    CheckBoxState State of a checkbox

    HorizontalAlignment Enumeration for horizontal alignment.
    MessageSeverity Message Severity enum.
    Orientation Enumeration of orientation.
    SortDirection Enumeration for sorting direction.
    TimeFormat Enumeration of time formats.
    VerticalAlignment Enumeration for vertical alignment.

    Name Description
    WaitSpinner Wait Spinner / Loader React component
