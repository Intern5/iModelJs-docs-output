    Toolbar

    Classes for working with a Toolbar

    Classes

    Name Description
    AppButton App button which displays icon.
    BackArrow Back arrow used in NestedGroup component.
    BackButton Back button which displays icon.
    Chevron Chevron component used in Indicator.
    Column Tool group column.
    Columns Columns of tool group.
    ExpandableButton Expandable toolbar button.
    ExpandableItem Expandable toolbar item.
    Expander Expandable entry of tool group panel.
    Group Tool group component.
    HistoryItem Basic history item used in Tray component.
    HistoryManager Helper to manage history entries.
    Icon History item with icon.
    Icon Toolbar item component that displays icon.
    Indicator Scroll indicator component.
    Item Toolbar item component.
    Items Toolbar items container.
    NestedGroup Nested tool group component.
    Overflow Expandable toolbar item component that displays ellipsis icon.
    Panel Basic panel.
    Scrollable A Toolbar with scroll overflow strategy.
    Title Tool group title.
    Tool Tool entry of tool group panel.
    Toolbar A toolbar that may contain items.
    ToolbarButton Basic toolbar button.
    ToolbarIcon Toolbar button which displays icon.
    Tray History tray used in ExpandableItem component.
