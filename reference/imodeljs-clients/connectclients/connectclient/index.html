    🔗ConnectClient Class

    Client API to access the connect services.

    🔗Methods

    Name Description
    ConnectClient ( deploymentEnv: DeploymentEnv ): ConnectClient Public
    delete ( token: AccessToken, relativeUrlPath: string ): Promise<void> Protected used by clients to send delete requests
    deleteInstance<T extends WsgInstance> ( token: AccessToken, relativeUrlPath: string, instance: T ): Promise<void> Protected used by clients to delete strongly typed instances through the standard WSG REST API
    getAccessToken ( authorizationToken: AuthorizationToken ): Promise<AccessToken> Public Gets the (delegation) access token to acess the service
    getDefaultUrl ( ): string Protected Gets the default URL for the service.
    getInstances<T extends WsgInstance> ( typedConstructor: object, token: AccessToken, relativeUrlPath: string, queryOptions: RequestQueryOptions ): Promise<T extends WsgInstance[]> Protected Used by clients to get strongly typed instances from standard WSG REST queries that return EC JSON instances.
    getInvitedProjects ( token: AccessToken, queryOptions: ConnectRequestQueryOptions ): Promise<Project[]> Public Get the projects the user has been "invited" to.
    getProject ( token: AccessToken, queryOptions: ConnectRequestQueryOptions ): Promise<Project> Public Gets a connect project.
    getProjects ( token: AccessToken, queryOptions: ConnectRequestQueryOptions ): Promise<Project[]> Public Gets connect projects accessible to the authorized user.
    getUrl ( excludeApiVersion: undefined | true | false ): Promise<string> Public Gets the URL of the service.
    getUrlSearchKey ( ): string Protected Gets name/key to query the service URLs from the URL Discovery Service ("Buddi")
    postInstance<T extends WsgInstance> ( typedConstructor: object, token: AccessToken, relativeUrlPath: string, instance: T extends WsgInstance, requestOptions: WsgRequestOptions ): Promise<T extends WsgInstance> Protected Used by clients to post strongly typed instances through standard WSG REST API
    postInstances<T extends WsgInstance> ( typedConstructor: object, token: AccessToken, relativeUrlPath: string, instances: T extends WsgInstance[], requestOptions: WsgRequestOptions ): Promise<T extends WsgInstance[]> Protected Used by clients to post multiple strongly typed instances through standard WSG REST API
    setupOptionDefaults ( options: RequestOptions ): Promise<void> Protected Augments request options with defaults returned by the DefaultWsgRequestOptionsProvider.
