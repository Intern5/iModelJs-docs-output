    🔗IModelHubClient Class

    Class that allows access to different iModel Hub class handlers. Handlers should be accessed through an instance of this class, rather than constructed directly.

    Briefcases ( ): BriefcaseHandler Public Get the handler for See: Briefcase related methods.

    GlobalEvents ( ): GlobalEventHandler Public Get the handler for See: IModelHubGlobalEvent related methods.

    setFileHandler ( fileHandler: FileHandler ): void Public Sets file handler for file upload/download.
