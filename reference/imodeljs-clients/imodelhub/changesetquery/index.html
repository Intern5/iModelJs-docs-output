    ChangeSetQuery Class

    Query object for getting ChangeSets You can use this to modify the query.

    see ChangeSetHandler.get

    Extends

    • InstanceIdQuery

    Methods

    Name Description
    afterVersion(versionId: string): this Query changeSets after version.
    betweenChangeSets(firstChangeSetId: string, secondChangeSetId?: string): this Query ChangeSets between changeSets.
    betweenVersionAndChangeSet(versionId: string, changeSetId: string): this Query changeSets between version and changeSet.
    betweenVersions(sourceVersionId: string, destinationVersionId: string): this Query changeSets between two versions.
    byId(id: string): this Query ChangeSet by its id.
    bySeedFileId(seedFileId: string): this Query changeSets by seed file id.
    fromId(id: string): this Query all ChangeSets that are after given ChangeSet id (does not include the specified change set)
    getVersionChangeSets(versionId: string): this Query version changeSets.
    latest(): this Change the order to latest changesets first in the query.
    selectDownloadUrl(): this Query will additionally select Briefcase file download URL.
