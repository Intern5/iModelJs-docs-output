    Code Code

    IModelHandler Handler for all methods related to See: IModel instances.
    IModelHubClient Class that allows access to different iModel Hub class handlers.
    IModelHubClientError Errors for incorrect iModel Hub requests.
    IModelHubError Error returned from iModel Hub service.
    IModelQuery Query object for getting iModels.

    InstanceIdQuery Query for instances with string based instance ids.
