    🔗VersionQuery Class

    Query object for getting Versions. You can use this to modify the query.

    🔗Extends

    • InstanceIdQuery

    🔗Methods

    Name Description
    addFilter ( filter: string, operator: "and" = undefined | "or" = undefined = "and" ): void Protected Add a part of the filter to currently set filter.
    byId ( id: string ): this Public Query single instance by its id.
    byName ( name: string ): this Public Query version by its name.
    filter ( filter: string ): this Public Set filter to the specified filter string.
    getId ( ): undefined | string Public Used by iModel Hub handlers to get the id that is queried.
    getQueryOptions ( ): RequestQueryOptions Public Method used by iModel Hub handlers to translate this query into request's QueryOptions.
    orderBy ( orderBy: string ): this Public Set order for the query.
    select ( select: string ): this Public Set select to specified select string.
    selectThumbnailId ( ...sizes: ThumbnailSize[] ): this Public Query will additionally select given sizes ids of thumbnails.
    skip ( n: number ): this Public Skip first entries in the query.
    top ( n: number ): this Public Select only top entries from the query.
