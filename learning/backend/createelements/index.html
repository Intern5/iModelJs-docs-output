    Creating Elements

    Use IModelDb.Elements.insertElement to insert a new element into an IModelDb. This method takes as its input an ElementProps or a subclass of it, which defines the class and all required properties of the new element.

    Example:

      public static insertRobot(iModelDb: IModelDb, modelId: Id64Props, name: string, location: Point3d): Id64Props {
    const props: GeometricElement3dProps = {      // I know what class and category to use.
      model: modelId,
      classFullName: RobotWorld.Class.Robot,
      category: Robot.getCategory(iModelDb).id,
      geom: Robot.generateGeometry(),
      placement: { origin: location, angles: new YawPitchRollAngles() },
      userLabel: name,
    };
    return iModelDb.elements.insertElement(props);
  }

    The above example shows the use of the GeometricElement3dProps subclass of ElementProps.

    Some classes provide create helper functions.

    Example:

          const cat: SpatialCategory = SpatialCategory.create(dictionary, _schemaNames.Class.Robot);
      cat.id = iModelDb.elements.insertElement(cat);
      cat.setDefaultAppearance(new SubCategoryAppearance({ color: ColorByName.silver }));
      iModelDb.elements.updateElement(cat);

    The above example shows the use of the SpatialCategory.create helper function to create the props object. It also illustrates how IModelDb.Elements.updateElement is used.
